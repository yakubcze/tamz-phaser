# TAMZ HRA

Hra vytvořená v JavaScriptu (pomocí frameworku Phaser) pro předmět Tvorba Aplikací pro Mobilní Zařízení.

Cílem hry je sesbírat co nejvíce mincí a výhybat se přitom bombám a nepříteli.
- při každé sebrané minci se objeví nová bomba s náhodnou rychlostí a náhodným směrem
- něpřítel se pohybuje přímočaře k postavě hráče
- nepřítel lze zastřelit pomocí mezerníku, ale při dalším sebrání mince se objeví znovu
- při kontaktu s nepřítelem se odečte 5 bodů a postava se vrátí na hřbitov
- při kontaktu s bombou nastane konec hry
